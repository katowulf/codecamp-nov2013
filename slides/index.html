What is Firebase?

  • A real-time data persistence platform
  • Provides authentication & security rules
  • Can serve as a complete backend (no servers required)

 

  • http://www.firebase.com/how-it-works-drawing-demo.html

Firebase ❤ Angular = AngularFire

<body ng-app="myModule">
  <div ng-controller="myApp">
    <input type="text" ng-model="name"/><br/><h1>Hello {{name}}!</h1>
  </div>
</body>
<script>
  angular.module("myModule", ["firebase"]).
      controller("myApp", function($scope, angularFire) {
         var url = "https://codecamp-nov2013.firebaseio.com/hello_example/";
         $scope.name = "";
         angularFire(new Firebase(url), $scope, "name");
      });
</script>
